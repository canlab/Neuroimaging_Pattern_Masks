# Individual-study maps

Result maps from individual published studies that aren't multivariate
signatures, atlases, or meta-analyses — typically Bayes-factor maps,
group-level contrast maps, or systems-component maps from large
single-study analyses. Each sub-folder has the standard layout
(`contents_description.md`, `visualize_contents.m`, data, PDF where
redistributable, `png_images/`).

## Sub-folders

| Year | Folder | Description |
| --- | --- | --- |
| 2024 | [Bo — Emotion-regulation Bayes factor (Nat Neurosci)](./2024_Bo_EmotionRegulation_BayesFactor/contents_description.md) | **Pilot** — four reappraisal-related system maps from a systems-identification approach using Bayes factors. |
| 2026 | [Miao — Social / ToM Bayes factor](./2026_Miao_Social-ToM_BayesFactor) | t-statistic + Bayes-factor maps for social vs. ToM contrasts across audio / text / conjunction modalities. |
