
Each study subfolder describes an atlas, parcellation, meta-analysis, multivariate signature, or group study map.  Each map is generally associated with a primary publication, and often with secondary publications.

Create Documentation for this repo, stored in docs/.  The main readme page should have file name and format compatible with auto loading in github. Don't remove existing content from files/folders, but reorganize it and incorporate it into a uniform plan.

For each study folder, create the following items.  Use any materials you find in the folder, including PDFs of the papers, notes, Readme files, and information at https://sites.google.com/dartmouth.edu/canlab-brainpatterns/home.
1. Download the PDF of the primary reference if it's not yet in the main folder, with file format <author last name>_<year>_<keywords>.pdf. Refer to it to describe the study and create other materials below.
2.  A Matlab script called visualize_contents showing (a) cortical surfaces, (b) isosurface if relevant, (c) montages of the image files for each image file in the study folder -- or, for combined atlases, the main atlas file in each folder. Use existing display/readme/visualization scripts from the various folders as examples to create a uniform plan.  Save .png files in  png_images/
3. A study-level markdown called contents_description.md that describes the contents of the folder. It will consist of:
- Overview of what the study is in a few lines, with reference to the primary paper and hyperlink to primary paper on publisher's site.  Also report citations to secondary papers if any are found in the folder.
- 1-2 Key images of the atlas and pattern, taken from https://sites.google.com/dartmouth.edu/canlab-brainpatterns/home or the Matlab script below
- How to load the atlas or image set from load_atlas() or load_image_set() in canlab core repo.
- Links to any scripts used to create the files, e.g., links to constructor scripts in the folder on canlab github repo online that build the atlas for composite atlases.
- A list of image files in the folder (generally .nii or .img) and/or atlas/signature .mat files and what each one is. Include relevant montage and surface for each file/atlas.
- Citations to other papers involved
