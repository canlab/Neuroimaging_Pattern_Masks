# Atlases and parcellations — cross-atlas summary materials

## Overview

This is a **utility folder** that holds cross-atlas summary materials
used to introduce or compare the various atlases distributed under
[`Atlases_and_parcellations/`](..). It is not itself an atlas and
does not provide a `load_atlas` keyword. Treat it as supplementary
documentation accompanying the per-atlas folders.

For the authoritative per-folder documentation, start at the top
level [`docs/README.md`](../../docs/README.md) of this repository,
which links to each atlas folder's `contents_description.md`.
