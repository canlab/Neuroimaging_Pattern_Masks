# Cerebellar template resources

## Overview

Cerebellar template and SPM-style **inter-template warp files** for
moving cerebellar data between the **MNI152NLin6Asym** whole-brain
space (FSL default, also the canlab2024 / canlab2023 cerebellum donor
space) and the **SUIT** cerebellar template of Diedrichsen 2006
*NeuroImage* (the standard space for cerebellum-only atlases like
Buckner-7, King 2019, MDTB). The actual warp / segmentation files live
under [`transforms/`](./transforms/) — there are no images at the top
level of this folder beyond the standard `.DS_Store`.

The warp set was generated by running the MNI152NLin6Asym 1 mm T1
through SPM12's `cereb` workflow (Diedrichsen SUIT toolbox) to produce
both an isolated cerebellum image and a SUIT-space deformation field;
the resulting files are used by CANlab atlases when the user requests
SUIT-space output.

## Primary reference

Derived from the MNI152NLin6Asym T1 template using the
[SUIT toolbox](https://www.diedrichsenlab.org/imaging/suit.htm) as
part of the CANlab build pipeline:

- Diedrichsen, J. (2006). *A spatially unbiased atlas template of the
  human cerebellum.* **NeuroImage, 33**(1), 127–138.
  [doi:10.1016/j.neuroimage.2006.05.056](https://doi.org/10.1016/j.neuroimage.2006.05.056)
- Diedrichsen, J., Balsters, J. H., Flavell, J., Cussans, E., &
  Ramnani, N. (2009). *A probabilistic MR atlas of the human
  cerebellum.* **NeuroImage, 46**(1), 39–46.
  [doi:10.1016/j.neuroimage.2009.01.045](https://doi.org/10.1016/j.neuroimage.2009.01.045)

## How to use

The deformation fields and segmentation outputs in
`transforms/MNI152NLin6Asym_T1_1mm_cerebellum_to_SUIT/` can be applied
with the SUIT toolbox or SPM12's batch GUI. To resample a
cerebellum-only NIfTI (already in MNI152NLin6Asym space) into SUIT
space, use the `u_a_*` deformation field with SPM `Normalise: Write`,
or `suit_reslice_dartel`.

The reverse direction (SUIT to MNI152NLin6Asym) can be obtained by
inverting the deformation field with SPM12 (`spm_invdef`) or by
chaining with the inter-template ANTs warps in
[`../transforms/`](../transforms/).

## File inventory

| File / dir | What it is |
| --- | --- |
| `transforms/MNI152NLin6Asym_T1_1mm_cerebellum_to_SUIT/` | SUIT-toolbox outputs warping the FSL T1 cerebellum into SUIT space (segmentation, isolation, deformation field). |
| `transforms/MNI152NLin6Asym_T1_1mm_cerebellum_to_SUIT/Affine_MNI152NLin6Asym_T1_1mm_seg1.mat` | Affine produced by SPM segmentation. |
| `transforms/MNI152NLin6Asym_T1_1mm_cerebellum_to_SUIT/MNI152NLin6Asym_T1_1mm_seg1.nii` | Grey-matter segmentation of the FSL T1. |
| `transforms/MNI152NLin6Asym_T1_1mm_cerebellum_to_SUIT/c_MNI152NLin6Asym_T1_1mm.nii` | SUIT-isolated cerebellum image. |
| `transforms/MNI152NLin6Asym_T1_1mm_cerebellum_to_SUIT/c_MNI152NLin6Asym_T1_1mm_pcereb.nii` | SUIT probabilistic cerebellum mask in template space. |
| `transforms/MNI152NLin6Asym_T1_1mm_cerebellum_to_SUIT/SUIT.nii` | Reference SUIT T1 template. |
| `transforms/MNI152NLin6Asym_T1_1mm_cerebellum_to_SUIT/u_a_MNI152NLin6Asym_T1_1mm_seg1.nii` | SUIT DARTEL flow field (MNI152NLin6Asym to SUIT). |
